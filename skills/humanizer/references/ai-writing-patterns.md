# AI writing patterns

Use this file when you need a deeper checklist while editing. Favor pattern families over exhaustive policing.

## Content inflation

1. Significance inflation
Use when text calls ordinary facts pivotal, enduring, transformative, or part of a broad historical shift without support.
Fix by stating the fact directly.

2. Notability name-dropping
Use when text lists major outlets or followers to imply importance.
Fix by keeping only the specific coverage that matters.

3. Superficial `-ing` analysis
Use when present-participle phrases pretend to add meaning: `highlighting`, `reflecting`, `showcasing`.
Fix by cutting the phrase or replacing it with a real fact.

4. Promotional language
Use when the tone sounds like tourism copy, PR, or ad copy.
Fix by replacing praise with neutral description.

5. Vague attributions
Use when claims hide behind `experts`, `observers`, or `reports`.
Fix by citing the real source if available or removing the attribution.

6. Formulaic challenge framing
Use when text inserts generic `despite challenges` or `future outlook` sections.
Fix by naming the actual problem or deleting the filler.

## Language habits

7. AI vocabulary
Watch for words like `additionally`, `testament`, `landscape`, `pivotal`, `vibrant`, `underscore`, and `showcase`.
Fix by choosing simpler words.

8. Copula avoidance
Use when text dodges `is` or `has` with phrases like `serves as`, `stands as`, `boasts`, or `features`.
Fix by using the plain verb if it is clearer.

9. Negative parallelism
Use when text says `it's not just X, it's Y`.
Fix by saying the point once.

10. Rule of three
Use when sentences force ideas into polished triples.
Fix by using the natural number of items.

11. Synonym cycling
Use when the same subject keeps changing labels only to avoid repetition.
Fix by repeating the clearest noun.

12. False ranges
Use when text says `from X to Y` for items that are not on a real scale.
Fix by listing the topics plainly.

## Style tells

13. Em dash overuse
Fix by using commas or periods unless the dash is doing real work.

14. Boldface overuse
Fix by removing mechanical emphasis.

15. Inline-header lists
Use when bullets begin with bold labels like `Performance:` or `Security:`.
Fix by converting to prose or a cleaner list.

16. Title case headings
Fix by using sentence case unless a style guide says otherwise.

17. Emoji decoration
Fix by removing emoji unless the audience clearly expects them.

18. Curly quotes
Fix only if the surrounding document uses straight quotes and consistency matters.

## Chatbot residue

19. Chatbot artifacts
Use when text includes `I hope this helps`, `certainly`, `would you like`, or other assistant framing.
Fix by deleting it.

20. Knowledge-cutoff disclaimers
Use when text says details are limited or current only to a training date.
Fix by removing the disclaimer and keeping only verified facts.

21. Sycophantic tone
Use when the text flatters the reader instead of answering.
Fix by responding directly.

## Filler and hedging

22. Filler phrases
Examples: `in order to`, `due to the fact that`, `at this point in time`.
Fix by shortening them.

23. Excessive hedging
Examples: `could potentially possibly`.
Fix by using one honest qualifier.

24. Generic conclusions
Use when text ends with `the future looks bright` or another empty upbeat wrap-up.
Fix by ending on a fact, a decision, or a concrete next step.

## Final audit prompt

After rewriting, ask:

`What still makes this sound obviously AI generated?`

Answer in a few bullets, then revise once more.
