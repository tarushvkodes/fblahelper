# FBLA Event Competition Format Guide

This file categorizes every event in `app.js` and summarizes what competitors face at competition.

## Sources Used (Web)

- FBLA National Competitive Events Operations Manual (2025-26): https://connect.fbla.org/headquarters/files/25-26-National-Competitive-Events-Operations-Manual.pdf
- Cumberland Valley FBLA event-type definitions: https://cvfbla.org/hs_events
- Georgia FBLA high school competitive events overview: https://georgiafbla.org/high-school-competitive-events/

## Format Key

- `Objective Test Event`: timed objective testing at conference.
- `Objective Test + Interactive Role Play`: objective testing plus judged scenario performance.
- `Objective Test + Parliamentary Demonstration`: objective/procedural knowledge plus judged parliamentary execution.
- `Presentation / Interview (No Objective Test)`: judged speaking/interview performance.
- `Production / Technical Performance (No Objective Test)`: judged technical build/performance artifacts.
- `Chapter / Report + Presentation (No Objective Test)`: chapter project/report and judged presentation outcomes.
